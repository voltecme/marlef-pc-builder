# Marlef x PC Builder Odoo Module

A powerful, open-source PC configurator for Odoo eCommerce, designed for Marlef Plaza.

## Features

- **Step-by-Step Configurator**: Intuitive UI for building custom gaming PCs.
- **Compatibility Engine**: Automatic filtering based on Socket, RAM type, and more.
- **AI 3D Visualization**: Integration support for converting PNG images to 3D models.
- **Backpanel Control**: Comprehensive management of components and user builds within Odoo.
- **Stock Sync**: Designed to synchronize with external providers like venus.tech.

## Installation

1. Clone this repository into your Odoo `addons` folder.
2. Update your Odoo configuration to include the path to this module.
3. Restart Odoo and update the App list.
4. Install the `Marlef x PC Builder` module.

## Usage

- **Backend**: Navigate to the "PC Builder" menu to manage components and view user builds.
- **Frontend**: Visit `/pc-builder` on your website to access the configurator.
